# E-commerce_data_analysis1

## Project Overview

This project is about analyzing e-commerce sales data to find out which products are making money and where the sales are coming from. I built a complete system that takes raw data, cleans it, and turns it into a beautiful dashboard.

## Tools I Used

MySQL: To store and join different data tables.

Python (Pandas): To clean the data and create new information.

Matplotlib & Seaborn: To make initial charts and find patterns.

Power BI: To create the final interactive dashboard.

## Step-by-Step Workflow

### Step 1: Database Management (SQL)

I started by importing two datasets (List of Orders and Order Details) into MySQL. I used the Inner Join command to combine them into one single table using the common column (Order ID). This helped me see all the information in one place.

### Step 2: Data Cleaning & Future Engineering (Python)

I connected Python to my SQL database and did the following:
Changed the date format so the computer can understand it correctly.
Created new columns like Order Month and Order Year to see sales trends over time.
Calculated the Profit Margin for every single sale to see which items are actually profitable.

### Step 3: Visual Analysis (Power BI)

Finally, I loaded the cleaned data into Power BI to build a dashboard. My dashboard shows:
Which categories (like Clothing or Electronics) have the most sales.
Monthly profit trends (to see which months are the busiest).
A map of sales across different states and cities.
